Este proyecto consiste en la creación de un juego simple con una libreria grafica ya preparada.

En la version normal el juego es "jugable". pudiendo:
  -Moverse.
  -Recolectar coleccionables.
  -Ganar la partida.

En el bonus, añado:
  -Animaciones simples.
  -Enemigos.
  -La posibilidad de perder.
